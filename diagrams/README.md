# Diagrams

This folder contains the database design diagrams for the Bright Sparks Ltd Inventory Management Database System.

## Contents

| File | Purpose |
|---|---|
| `ERD.png` | Final Entity Relationship Diagram showing the database structure and table relationships. |
| `UNF.png` | Unnormalised data structure from the original scenario. |
| `1NF.png` | First Normal Form design. |
| `2NF.png` | Second Normal Form design. |
| `3NF.png` | Third Normal Form design. |

## Notes

The diagrams show how the database design progressed from an unnormalised table into a relational database structure.

The ERD is the main design reference. The normalisation diagrams show the process used to reduce repeated data and improve data integrity.
