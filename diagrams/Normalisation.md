# Normalisation

The database design was normalised from the original unnormalised data table through to Third Normal Form.

## UNF

The original data contained repeated values, combined fields and multiple pieces of information stored together.

## 1NF

The first step separated repeating groups so each field stored a single value.

## 2NF

The second step removed partial dependencies and separated data into more focused tables.

## 3NF

The final step reduced unnecessary dependencies and improved the overall database structure.

## Result

The final design reduced duplicated data and made the database easier to maintain.
